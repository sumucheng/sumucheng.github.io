浏览器渲染过程

  1. 根据 HTML 构建 HTML 树（DOM）
  2. 根据 CSS 构建 CSS 树（CSSOM）
  3. 将两棵树合并成一颗渲染数（render tree）
  4. Layout 布局（文档流、盒模型、计算大小和位置）
  5. Paint 绘制（颜色、阴影等）
  6. Composite 合成（根据层叠关系展示画面）

更新样式

  • 使用 JS 更新样式，例：
    • div.style.background=‘red’ //修改样式
    • div.style.display=‘none’ //修改样式
    • div.classList.add(‘red’) //添加类
    • div.remove() //删除节点

更新方式

  • JS/CSS>样式>布局>绘制>合成
    • 例如 div.remove()
  • JS/CSS>样式>绘制>合成
    • 例如改变背景颜色
  • JS/CSS>样式>合成
    • 例如改变 transform

CSS 动画优化

  • JS 优化
    • 使用 requestAnimationFrame 代替 setTimeout 或 setInterval
  • CSS 优化
    • 使用 will-change 或 translate

transform

  • translate 位移
  • scale 缩放
  • rotate 旋转
  • skew 倾斜
  • 一般都需要配合 transition 过渡
  • inline 元素不支持 transform，需要先变成 block
  • skew 用得较少
  • 可以组合使用，例如 transform:scale(0.5) translate(-10%,-10%);
  • 更多信息：https://developer.mozilla.org/zh-CN/docs/Web/CSS/transform

transform:translate

  • transform: translate(12px, 50%);
  • transform: translateX(2em);
  • transform: translateY(3in);
  • transform: translateZ(2px); (父容器需要设置 transform:perspective(17px))
  • transform: scale3d(2.5, 1.2, 0.3);

translate(-50%,-50%)可使绝对定位元素居中

transform:scale

  • transform: scale(2, 0.5);
  • transform: scaleX(2);
  • transform: scaleY(0.5);

transform:rotate

  • transform: rotate(0.5turn);
  • transform: rotateX(10deg);
  • transform: rotateY(10deg);
  • transform: rotateZ(10deg);

一般用于 360 度旋转制作 loading

transform:skew

  • transform: skew(30deg, 20deg);
  • transform: skewX(30deg);
  • transform: skewY(1.07rad);

transition

  • 补充中间帧
  • transition: 属性名 过渡市场 过渡方式 延迟时长
  • 可以用逗号分隔两个不同的属性
  • 可以用 all 表示所有属性
  • 过渡方式：linear | ease | ease-in | ease-out | ease-in-out

不是所有属性都可以过渡

  • display:none=>block 无法过渡，一般使用 visibility:hidden=>visible
  • 过渡必须要有起始

animation

  1. 声明关键帧

    @keyframes slidein{
from{
  transform:translateX(0%);
}
to{
  transform:translateX(100%);
}
}
@keyframes indentifier{
0%{top:0;left:0;}
30%{top:50px;}
100%{top:100px;left:100%;}
}

  2. animation

  • 时长 : 1s | 1000ms
  • 过渡方式
  • 延迟时长
  • 次数 : 3 | 2.4 |infinite
  • 方向 : reverse | alternate | alternate-reverse
  • 填充模式 : none | forwards | backwards | both
  • 是否暂停 : paused | running
  • 动画名
