使用hugo搭建个人博客

October 4, 2019

hugo 简介

go 语言实现的一个博客生成器

安装 hugo

  • mac
    • brew install hugo
    • hugo version
  • windows
    • 下载安装包 https://github.com/gohugoio/hugo/releases
    • 解压
    • 将 hugo.exe 所在路径添加到 PATH
    • 重启终端，运行 hugo version 查看是否安装成功

使用 hugo 搭建博客

官网教程：https://gohugo.io/getting-started/quick-start/

  • step1: 创建一个新网站

    hugo new site quickstart

  • step2: 添加主题

    //下载主题
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
//添加ananke主题
echo 'theme = "ananke"' >> config.toml

  • step3: 添加一些内容

    hugo new posts/my-first-post.md

  • step4: 启动 hugo 服务器

    // 预览草稿
hugo server -D
// 预览非草稿
hugo server

  • step5: 自定义主题

    //在编辑器中打开 config.toml, 并修改
baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

  • step6: 建立静态页面

    hugo
