CSS 的历史

版本 时间 描述
CSS1 1996 年
CSS2 1998 年 添加了定位、z-index、media
CSS2.1 2004~2011 年 使用最广泛的版本，几乎所有浏览器都支持 100%的特性
CSS3 1999 年 现代版本，各模块分开制定，IE8 部分支持
CSS4* 各模块各自升级

查询某浏览器是否支持某特性

使用 caniuse.com

CSS 的语法

语法 1：样式语法

selector{
    property_name: property_value;
    /*comment*/
}

注意事项

  • 区分大小写
  • 浏览器不会报错
  • 最后一个分号可以省略，但建议不要省略

语法 2：@语法

@charset "UTF-8"; /*字符编码的声明必须在第一行*/
@import url(1.css); /*导入1.css文件*/
@media (min-width:100px) and (max-width:200px){
    ...
}/*媒体查询*/

注意事项

  • charset 表示字符集
  • UTF（Unicode Transformation Format）是针对 Unicode 字符集的一种字符编码，常见的 UTF 格式有：UTF-7, UTF-7.5, UTF-8,UTF-16, UTF-32

border 调试法

selector{
    ...
    border:1px solid red;
    ...
}
