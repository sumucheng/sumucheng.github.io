基本概念

  • 文档流 Normal Flow
  • 块（block）、内联（inline）、内联块（inline-block）
  • 两种盒模型
  • margin 合并

文档流

  • 流动方向

    • inline 元素从左到右，到达最右才会换行，一个元素可以跨行
    • block 元素从上到下，每一个元素都另起一行
    • inline-block 元素也是从左到右，但一个元素不会跨行

  • 宽度

    • inline 宽度为内部 inline 元素的和，不能用 width 指定
    • block 默认自动计算宽度（不等于 100%），可以用 width 指定
    • inline-block 宽度为内部 inline 元素的和，但可以用 width 指定

  • 高度

    • inline 高度由 line-height 间接确定，和 height 无关
    • block 高度由内部文档流元素决定（无元素时高度为 0），可以设置 height
    • inline-block 与 block 类似，可以设置 height

  • 脱离文档流

    • position:absolute/fixed;
    • float:;

注意事项

  • 不要在 inline 元素中插入 block 元素
  • 尽量不要把宽度设置成 100%

当内容大于容器时，使用 overflow（溢出）

  • overflow:visible: 显示溢出部分
  • overflow:hidden: 隐藏溢出部分
  • overflow📜 显示滚动条
  • overflow:auto：当内容大于容器时，显示滚动条，否则不显示

盒模型 Box Model

  • CSS 盒模型（box model）本质上是一个封装周围 HTML 元素的盒子，它包括:
    • 边距 margin
    • 边框 border
    • 填充 padding
    • 内容 content
  • CSS3 引入了 box-sizing 属性来设置盒模型的类型，包括三个取值： content-box，padding-box 和 border-box
  • content-box 内容盒
    • 内容就是盒子的边界
    • width/height = content 宽度/高度
  • border-box 边框盒
    • 框架才是盒子的边界
    • width/height = content 宽度/高度 + padding + border

margin 合并

  • 只有上下方向存在 margin 合并，左右方向不合并

  • 取消父子的 margin 合并

    • 设置 border/padding
    • overflow : hidden;
    • display : flex;

  • 取消兄弟的 margin 合并

    • 使用 inline-block

基本单位

  • 长度单位
    • px 像素
    • em 相对于自身 font-size 的倍数
    • %
    • 整数
    • rem
    • vw/vh
  • 颜色
    • #FFFFFF/#FFF //十六进制
    • rgb(0,0,0) / rgba(0,0,0,1)
    • hsl(360,100%,100%) //色相，饱和度，亮度
