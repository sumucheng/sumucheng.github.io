HTML入门笔记1

October 6, 2019

HTML 的历史

  • 1990 年左右，Tim Berners-Lee 爵士发明了初版 HTML
  • 1993 年 6 月，IETF（互联网工程工作小组）工作小組发布第一版草案
  • 1994 年 10 月，W3C 成立，HTML 标准规范交由 W3C 协会制定
  • 1995 年 11 月，HTML2.0 发布
  • 1996 年 1 月，HTML3.2 发布
  • 1997 年 11 月，HTML4.0 发布
  • 2000 年 1 月，XHTML1.0 发布
  • 2001 年 5 月，XHTML1.1 发布
  • 2014 年 10 月，HTML5.0 发布

HTML 第一行

<!DOCTYPE html>

常用的章节标签

标签 描述
h1~h6 标题
div 块级区域
span 内联区域
section 章节
article 文章内容
header 头部内容
footer 底部内容
main 主要内容
aside 侧边栏

常用的内容标签

标签 描述
ol 有序列表
ul 无序列表
li 列表项
dl 描述列表
dt 描述列表中的项目（term）
dd 描述列表中的项目描述
pre 预格式文本（保留空格和换行，文本表示为等宽字体）
br 换行
hr 水平分割线
a 链接
em 强调文本（斜体）
strong 强调文本（粗体）
i 斜体文本
b 粗体文本
code 代码文本
q 内联引用
blockquote 块引用

常用的全局属性

标签 描述
class 元素类名
id 元素唯一 ID
style 元素样式
title 元素额外信息
hidden 隐藏元素
tabindex Tab 键控制次序
contenteditable 内容可编辑
lang 元素的语言代码
spellcheck 检查拼写错误
translate 翻译元素的值
