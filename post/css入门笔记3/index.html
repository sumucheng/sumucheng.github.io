CSS入门笔记3 - 布局

October 9, 2019

布局方式

  • 固定宽度布局：一般宽度为 960 / 1000 / 1024 px
  • 不固定宽度布局：主要依靠文档流的原理布局
  • 响应式布局：PC 上固定宽度，移动端不固定宽度

布局类型

  • float 布局
  • flex 布局
  • grid 布局

float 布局

  • 子元素添加 float 和 width 属性

  • 父元素添加.clearfix 类

    .clearfix:after {
content: "";
display: block;
clear: both;
}

注意事项

  • 在宽度上可以留一些空间，或者每行最后一个元素不设 width
  • 不需要做响应式，因为手机上没有 IE
  • 宽度固定的块元素的左右 margin 设为 auto 会使其居中
  • margin-left:auto; margin-right:auto; 比 margin:0 auto 要好
  • 负 margin
  • float 布局的缺点：需要程序员自己计算宽度，不灵活，无法适应手机页面

flex 布局

  • flex container 的样式

    /*让一个元素变成 flex 容器*/
display: flex;
/*改变主轴 item 的流动方向*/
flex-direction: row | row-reverse | column |column-reverse;
/*设置是否自动换行*/
flex-wrap: nowrap | wrap | wrap-reverse;
/*主轴对齐方式*/
justify-content: flex-start |
flex-end|center|space-between|space-around|space-evenly;
/*次轴对齐方式*/
align-items: flex-start | flex-end|center|stretch|baseline;

  • flex item 的样式

    • order
    • flex-grow
    • flex-shrink
    • flex-basis
    • align-self:定制 align-items

注意事项

  • 尽量不把 width 和 height 写死，适应 min-width/height 和 max-width/height
  • 使用 margin-xxx:auto 可以达到类似 space-between 的效果

grid 布局

  • grid container 样式

    .container {
display: grid;
grid-template-columns: 40px 50px auto 50px 40px;
grid-template-rows: 25% 100px auto;
}
.container {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
grid-template-rows: 1fr 2fr;
}

  • grid item 样式

    .item {
grid-row-start: 1;
grid-row-end: 2;
grid-column-start: 1;
grid-column-end: 3;
}

  • grid
