使用github pages预览HTML

October 5, 2019

创建个人 github 仓库

仓库名设为：[github 用户名].github.io

创建本地仓库

cd public
git init
git add .
git commit

关联远程 github 仓库

git remote add origin git@github.com:[github 用户名]/[github 用户名].github.io.git
git push -u origin master

打开 HTML

  • 进入[github 用户名].github.io
  • 进入 setting
  • 打开 github pages
