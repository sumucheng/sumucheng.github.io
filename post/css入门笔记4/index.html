position

  • static
  • relative
  • absolute
  • fixed
  • sticky

position:static

  • 默认值，仍处于文档流

position:relative

  • 使用场景：给 absolute 元素做父元素
  • 不脱离文档流

position:absolute

  • 使用场景：脱离原来的场景，比如对话框的关闭按钮、鼠标悬浮提示框
  • 定位基准是祖先元素中最近的非 static 元素，通常在父元素设置 relative
  • 某些浏览器上如果不写 top/bottom 和 left/right 会导致位置错乱
  • left:100%
  • left:50% + 负 margin

position:fixed

  • 使用场景：回到顶部按钮
  • 手机上尽量不用

position:sticky

  • 使用场景：导航栏
  • 兼容性较差

层叠上下文

  • 非 static 元素在所有 static 元素的上方或下方
  • 每个层叠上下文是独立的作用域，z-index 只能在同一作用域中比较
  • 创建层叠上下文的方法
    • z-index 不为 auto 的绝对/相对定位
    • 固定（fixed） / 沾滞（sticky）定位
    • opacity 值小于 1
    • transform 属性不为 none
    • flex
    • 更多：https://developer.mozilla.org/zh-CN/docs/Web/Guide/CSS/Understanding_z_index/The_stacking_context
