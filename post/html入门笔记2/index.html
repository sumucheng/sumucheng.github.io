HTML入门笔记2 — 常用标签

October 7, 2019

<a>标签

作用

  • 跳转到外部页面
  • 跳转内部锚点
  • 跳转到邮箱或电话

属性

  • href：指定链接的目标
  • target：规定在何处打开目标
  • download：指定下载链接
  • rel=noopener

href

  • 网址：

    • http://
    • https://
    • //

  • 路径

    • /a/b/c
    • a/b/c
    • index.html
    • ./index.html

  • 伪协议

    <a href="javascript:;">空的伪协议，点击链接后页面不会刷新</a>

  • 跳转到邮箱或电话

    <a href="mailto:772765729@qq.com">发送邮件</a>
<a href="tel:17317824743">打电话</a>

target

  • _self（默认值）：当前页面打开
  • _blank：新页面打开
  • _top：顶层窗口打开
  • _parent：父级窗口打开
  • window的name
  • iframe的name

<img>标签

作用

  • 发出get请求，展示一张图片

属性

  • src：规定图片的URL
  • alt：定义图片的替代文本
  • height/width：定义图片的高度/宽度（同时定义可能造成图片变形）

事件

  • onload：当图片加载成功时运行脚本

  • onerror：当图片加载失败时运行脚本

    <script>
xxx.onload = function() {
    console.log('图片加载成功')
}
xxx.onerror = function() {
    console.log('图片加载失败')
    xxx.src = "/404.png"
}
</script>

响应式

<style>
    img{
        max-width: 100%;
       }
</style>

<table>标签

相关标签

  • table
  • thead：表头
  • tbody：主体
  • tfoot：页脚
  • tr：定义表格行
  • th：定义表头
  • td：定义表格单元

样式

  • table-layout(auto/fixed)
  • border-collapse：合并边框
  • border-spacing：边框间距

<form>标签

作用

  • 发出get或post请求，然后刷新页面

属性

  • action：规定提交表单时向何处发送表单数据
  • method：规定用于发送表单数据的HTTP方法
  • target：规定在何处打开action URL
  • autocomplete（on/off）：自动完成

事件

  • onsubmit：当提交表单时运行脚本

注意事项

  • form里面的input要有name
  • form里面需要含有type=submit才可以触发submit事件

<input>标签

作用

  • 让用户输入内容

属性

  • type
  • name
  • autofocus
  • checked
  • disabled
  • value
  • placeholder

type

type 类型
text 单行文本
password 密码
button 按钮
file 上传文件
radio 单选
checkbox 多选
submit 提交按钮
hidden 隐藏输入

事件

  • onchange：当元素改变时运行脚本
  • onfocus：当窗口获得焦点时运行脚本
  • onblur：当窗口失去焦点时运行脚本

验证器（HTML5新增功能）

其他输入标签

  • select+option：下拉选择
  • textarea：长文本
  • lable：相关标记
